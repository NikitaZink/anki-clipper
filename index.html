Вопросы

Часто задаваемые вопросы

Да, расширение полностью бесплатно. Groq предоставляет бесплатный API с щедрыми лимитами. Ollama работает локально без каких-либо затрат.

При использовании Ollama текст обрабатывается исключительно на твоём устройстве. При Groq текст передаётся в их API (политика конфиденциальности Groq).

Расширение создано для Яндекс Браузера, но совместимо с любым Chromium-браузером (Chrome, Edge, Brave).

Бесплатный плагин для Anki, который позволяет внешним программам добавлять карточки через API. Устанавливается за 30 секунд.

Установка

Установка за 3 шага

1
Установи расширение из магазина Яндекс Браузера
2
Установи AnkiConnect в Anki Код плагина: 2055492159
3
Укажи API-ключ Groq в настройках расширения
