Политика конфиденциальности

Дата обновления: 26 мая 2026 года

Anki Clipper — браузерное расширение для создания карточек Anki из текста. Эта политика объясняет, какие данные использует расширение и куда они могут передаваться.

Какие данные использует расширение

  • Текст, который пользователь вручную копирует в буфер обмена и затем отправляет на обработку в расширении.
  • Сгенерированные карточки, история сессий и пользовательские настройки.
  • API-ключи и адреса выбранных AI-провайдеров, если пользователь сохраняет их в настройках расширения.
  • Название выбранной колоды Anki и технические данные, необходимые для подключения к AnkiConnect.

Как используются данные

Данные используются только для основных функций расширения: анализа текста, генерации карточек, редактирования результата и отправки карточек в локальное приложение Anki через AnkiConnect.

Передача данных третьим лицам

Разработчик расширения не продает пользовательские данные и не передает их для рекламы, кредитования или иных целей, не связанных с работой расширения.

Если пользователь выбирает облачный AI-провайдер, например Groq или OpenAI-совместимый API, выбранный текст и служебный запрос отправляются этому провайдеру для генерации карточек. Обработка таких данных регулируется политикой конфиденциальности выбранного провайдера.

Если пользователь выбирает Ollama, обработка выполняется на локальном адресе, указанном в настройках. Данные также могут отправляться на локальный AnkiConnect по адресу http://localhost:8765 для добавления карточек в Anki.

Хранение данных

Настройки расширения и API-ключи сохраняются средствами Chrome Storage. История сессий и временные данные сохраняются локально в браузере пользователя. Разработчик не имеет доступа к этим данным.

Удаление данных

Пользователь может удалить данные расширения, очистив историю внутри расширения, изменив настройки, удалив API-ключи или удалив расширение из браузера.

Контакты
